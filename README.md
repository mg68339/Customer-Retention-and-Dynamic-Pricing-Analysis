# Omnichannel Retail Sales & Inventory Analytics Dashboard

## Project Overview

This project analyzes retail sales data to identify trends, optimize inventory, and generate business insights. The goal is to build an interactive dashboard for better decision-making.

## Objectives

* Analyze sales performance
* Identify top products and categories
* Track sales trends over time
* Analyze sales by region/city
* Identify peak sales periods

## Tech Stack

* Python (Pandas)
* SQL
* Power BI / Tableau
* Excel

## Project Structure

* data/ → datasets
* notebooks/ → Python analysis
* sql/ → SQL queries
* dashboard/ → dashboard files
* reports/ → final report
